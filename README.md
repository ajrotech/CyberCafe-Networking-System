# Cyber Cafe Networking System

This project demonstrates a **Cyber Cafe Networking System** designed using **Cisco Packet Tracer** for the Computer Networks course at **Sukkur IBA University**.

**Instructor:** Engr. Muhammad Younis Mughal

## Project Files
- `CyberCafe.pkt` – Cisco Packet Tracer file
- `Report.pdf` – Project report

## Description
The project covers network setup, IP addressing, device configuration, and resource management to efficiently connect multiple computers in a small-scale network.
